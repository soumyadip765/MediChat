# 🩺 MediChat – Health Support Assistant

MediChat is an AI-powered healthcare chatbot designed to provide general health information, symptom guidance, and self-care suggestions through a real-time conversational interface. It integrates rule-based logic with generative AI to deliver safe, structured, and user-friendly responses.

---

## 🚀 Features

- 💬 Real-time chat interface with responsive UI  
- 🧠 Hybrid response system (rule-based + AI-generated)  
- ⚠️ Emergency detection with instant alert and helpline guidance  
- 📋 Predefined quick-response modules for common symptoms  
- 🔒 Safety-focused responses (no diagnosis, only guidance)  
- 🔄 Seamless frontend-backend communication via REST API  


---

## 🛠️ Tech Stack

- **Backend:** Python, FastAPI  
- **Frontend:** HTML, CSS, JavaScript  
- **AI Integration:** Google Gemini 2.5 Flash API  
- **Architecture:** REST API, Modular Design  

---

⚠️ Disclaimer

MediChat provides general health information only and is not a substitute for professional medical advice, diagnosis, or treatment. Always consult a qualified healthcare provider for medical concerns.
